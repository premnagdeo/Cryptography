# Cryptography
A collection of cryptographic techniques written in Python3.


## Ciphers Implemented
- Reverse Cipher
- Caesar Cipher
- Rot 13 Cipher
- Transposition Cipher


## Requirements:

> Python 3.x


## Dependencies:

> pyperclip - https://github.com/asweigart/pyperclip

For copying plaintext/ciphertext to clipboard.


## Usage

Run the Python Program
> python  <file_name>.py
